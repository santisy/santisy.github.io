Display Item List

Dispaly Item List is for manipulating the display items. You can do actions like Drag, Hide, Show All, Delete on items. For dragging, please directly use your computer mouse to drag them. For the later three actions, click the right Actions button. NOTE that the Delete action will delete items from the remote server, so please use Delete when only necessary.
