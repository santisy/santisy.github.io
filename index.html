This is a Dispaly Item List for manipulating the display items. You can do actions like Drag, Hide, Show, Delete on items. For dragging, please directly use your computer mouse to drag them. For the later three actions, click the Actions button. NOTE that
1. the Delete action will delete items from the remote server, so please use Delete when only necessary.
2. Hide will also hide images in magnifying and per-image displaying mode.
